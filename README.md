## Fashion Detection
The project involves implementing an object detection model based on neural networks to identify and classify clothing items in images, using a dataset annotated with bounding boxes to train and evaluate the model.

### Dataset
Download [Colorful Fashion Dataset For Object Detection](https://www.kaggle.com/datasets/nguyngiabol/colorful-fashion-dataset-for-object-detection), unzip and paste the entire file so you can run the project without any directory issues.
